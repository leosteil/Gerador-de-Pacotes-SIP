Gerador de Pacotes SIP

Buscando aplicar os conceitos vistos nas aulas da disciplina de Bases da Gestão Eletrônica de Documentos, este trabalho será desenvolvido para a criação de pacotes SIP (primeiro estágio da cadeia de custódia de Documentos Arquivisticos Digitais).

Primeiramente o trabalho será focado na criação de pacotes SIP para Documentos nato-digitais, ou seja documentos que nascem em meio digital. Posterior a isso, pretende-se ampliar o sistema para adquirir novas funcionalidades.

O modelo de metadados seguido é o DoublinCore com 15 metadados.
